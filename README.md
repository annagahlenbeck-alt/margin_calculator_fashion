# margin_calculator_fashion (DTC + Wholesale)

A lightweight Python tool for fashion entrepreneurs to calculate accurate product prices for both wholesale and DTC markets.

The calculator:
- Computes production cost (materials + labour + other expenses)
- Uses industry-typical multipliers (2.2–2.5 for wholesale, 4.0–5.5 for DTC)
- Compares your prices to competitor benchmarks
- Generates a suggested DTC and wholesale retail price

## Features
- Clean, modular Python functions
- User input for production cost and competitor prices
- Automatic averaging of competitor pricing
- Combined margin + market-based DTC calculation
